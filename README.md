- **飞行动力学原理**：概述飞行器动力学的基本理论，帮助读者构建理论基础。
- **模型定制**：教授如何创建或调整飞机模型，包括控制表面、发动机性能参数等。
- **脚本与控制逻辑**：介绍如何编写脚本来控制飞行测试或模拟特定飞行条件。
